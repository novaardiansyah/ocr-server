# Pengenalan

- Node with Express.js
- OCR

# Instruksi

Kerjakan setiap bagian dari taks yang tertera pada file `.agent/.docs/task.md` (WAJIB ABAIKAN BAGIAN LAIN), setiap task memiliki scope/workspace yang membatasi jadi hanya modifikasi/kerjakan sesuai dengan batasan, dan jika ada dokumentasi/planning maka buat file markdown dan simpan ke direktori `.agent/.docs/*`.

Setiap poin yang terselesaikan pada `.agent/.docs/task.md`, maka beri tanda (DONE) di akhir poin, dan kamu TIDAK PERLU memberi respon/penjelasan apapun terkait perubahan.

WAJIB: kode program harus bersih dari komentar, better align (tab sejajar bila perlu), dan space/tab size=2.

# Batasan

Perintah seperti `npm run dev` atau `npm run build` tidak di izinkan, segala test akan dilakukan manual.
